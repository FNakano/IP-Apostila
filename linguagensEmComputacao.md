# Linguagens usadas em computação - uma explicação curta

Cronologicamente (ou, historicamente), linguagens de programação (https://en.wikipedia.org/wiki/Programming_language) foram criadas antes de outras linguagens, por exemplo as de marcação (https://en.wikipedia.org/wiki/Markup_language
). A medida em que o uso de computadores foi aumentando, novas linguagens foram sendo criadas (mesmo que, em princípio, não fossem consideradas linguagens. Com essa evolução cronológica, os termos *linguagem de computação* e *linguagem de programação*, outrora considerados sinônimos, deixaram de sê-lo (https://en.wikipedia.org/wiki/Computer_language). 

> A computer language is a formal language for humans to communicate with a computer, not a natural language. In earlier days of computing (before the 1980s), the term was used interchangeably with programming language, but today, used primarily for taxonomy, is a broader term that encompasses languages that are not programming in nature. Sub-categories (with possibly contended hierarchical relationships)

Acrescentando a informação que há linguagens de programação parecidas e outras diferentes entre si, por exemplo, C é parecido com C++ que é parecido com Java; PHP é parecido com BASIC; LISP é (muito) diferente de C; Prolog é (muito) diferente de C, ...
é possível introduzir uma nova caracterização. Esta é nomeada *paradigma de programação* (https://en.wikipedia.org/wiki/Programming_paradigm).

Como existem linguagens multi-paradigma, uma taxonomia de linguagens, à semelhança de uma árvore genealógica, não é possível sem que se façam concessões.

De acordo com essas categorizações, tem-se:
  
> C is an imperative procedural language, supporting structured programming, lexical variable scope, and recursion, with a static type system. It was designed to be compiled to provide low-level access to memory and language constructs that map efficiently to machine instructions, all with minimal runtime support. Despite its low-level capabilities, the language was designed to encourage cross-platform programming. A standards-compliant C program written with portability in mind can be compiled for a wide variety of computer platforms and operating systems with few changes to its source code. (https://en.wikipedia.org/wiki/C_(programming_language))
