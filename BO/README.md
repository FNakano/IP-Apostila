Orientações:

O e-disciplinas controla/mantém versões das entregas realizadas. Desta forma, versões melhores das respostas podem ser enviadas, se desejado.

A fim de saber quanto tempo é usado para entregar que parte da tarefa, ela foi planejada para ser difícil de completar dentro do tempo da aula de laboratório.

A atividade pode ser feita em duplas, desde que os dois cuidem de aprender. Não se recomenda grupos maiores.

A avaliação nessa atividade não compõe a nota na disciplina mas pode ser um indicador de aproveitamento.

O programa a ser entregue deve ser uma implementação do algoritmo de ordenação apresentado no briefing em sala de aula feito na manhã de 19/06. Em poucas palavras: "implementar o algoritmo de ordenação encapsulando em uma função o algoritmo de busca e usando esse algoritmo de busca (ié chamando a função), conforme explicado no briefing, para ordenar."

O array pode ter elementos do tipo int, float ou double, como preferir.

As funções podem usar variáveis de escopo global, o que evita passar parâmetros. Por outro lado, nesse caso, pode ser conveniente passar parâmetros. Cabe a você definir e implementar.

Pelo menos uma entrega deve ser feita durante a atividade de laboratório. O arquivo contendo o código-fonte deve ser entregue como anexo, o tamanho é limitado em 10kB. Caso mais de uma pessoa tenha escrito o programa, as outras pessoas devem informar no texto da resposta quem foi o colega que entregou o arquivo.

Sugestão para construção do programa (seguir esses passos é opcional)

- Teste com arrays de diferentes comprimentos (tenha várias definições de arrays e comente/descomente conforme precisar)

- Inicie do "olá mundo" (escreva o programa e o execute);

- Inclua a criação de um array, teste a execução;

- Inclua em main comandos para imprimir os elementos desse array, teste a execução;

- Encapsule os comandos que imprimem o array em uma função para imprimir os elementos do array, teste a função;

- Inclua em main comandos para buscar um elemento nesse array, teste a execução;

- Encapsule os comandos que buscam um elemento no array em uma função para buscar elementos do array, teste a função;

- Inclua em main comandos que usam a função de busca e ordenam o array, teste a execução;

- Troque o array (conteúdo e tamanho e teste a execução;
